# 📖 Azure Free vs. Paid Guide

This guide helps you understand how Azure billing works. Follow these rules to keep your account safe from unexpected charges.

---

## 🟢 1. Things That Are Free Until Account Closure

These services cost ₹0 forever. You can use them as long as you stay under the monthly limits.

| Service | Description |
|---------|-------------|
| **App Registrations** | Creating app IDs and secret keys is always free |
| **Azure Active Directory (Entra ID)** | Managing users and logins costs nothing |
| **Azure Functions** | You get 1 million free code requests every single month |
| **Azure App Service** | You can host up to 10 small web apps for free |
| **Cosmos DB** | You get 25 GB of free database storage |
| **Azure DevOps** | Up to 5 people can build code together for free |
| **Azure Advisor** | Free tips to help secure your cloud setup |
| **Billing Support** | Asking Microsoft billing questions is always free |

---

## 🔴 2. Things That Cost Money (What is Charged)

These items will eat up your free credits. If your free trial ends and you upgrade, these will charge your credit card directly.

| Service | Description |
|---------|-------------|
| **Virtual Machines (VMs)** | Running big computer servers in the cloud |
| **SQL Databases** | Storing data outside of the free database tiers |
| **Public IP Addresses** | Buying a fixed web address for your servers |
| **Data Transfers** | Moving massive amounts of data out of Azure to the internet |
| **Unused Disks** | If you delete a virtual server but forget to delete its hard drive, you still pay for the storage space |

---

## 🛡️ 3. Safety Steps to Prevent Big Bills

Follow these basic habits to keep your costs at exactly ₹0:

- **Watch the Blue Banner:** Never click the "Upgrade now" button on your billing page unless you are ready to pay real money.

- **Use Resource Groups:** Put all your practice projects inside one single Resource Group folder. When you are done studying, delete the entire folder. This wipes out everything inside so nothing runs in the background.

- **Turn Off Virtual Tools:** If you build a virtual machine to test it, shut it down when you go to bed.

- **Use Learn Sandboxes:** Use the free lab environments inside Microsoft Learn documents. They automatically shut off after 4 hours and never link to your credit card.
